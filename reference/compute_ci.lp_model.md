# Compute confidence interval for local polynomial model

Compute confidence interval for local polynomial model

## Usage

``` r
compute_ci.lp_model(x, ci = 0.95, ...)
```

## Arguments

- x:

  \- serosv models

- ci:

  \- confidence interval

- ...:

  \- arbitrary arguments

## Value

confidence interval dataframe with 4 variables, x and y for the fitted
values and ymin and ymax for the confidence interval
