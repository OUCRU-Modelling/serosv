# Plot output for corrected_prevalence

Plot output for corrected_prevalence

## Usage

``` r
plot_corrected_prev(x, y = NULL, facet = FALSE)
```

## Arguments

- x:

  \- the output of \`correct_prevalence()\` function

- y:

  \- another output of \`correct_prevalence()\` function (optional, for
  comparison only)

- facet:

  \- whether to plot as facets or on the same plot (only when y is
  provided)

## Value

ggplot object
