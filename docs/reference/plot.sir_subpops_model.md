# plot() overloading for SIR sub populations model

plot() overloading for SIR sub populations model

## Usage

``` r
# S3 method for class 'sir_subpops_model'
plot(x, ...)
```

## Arguments

- x:

  the sir_subpops_models object.

- ...:

  arbitrary params.

## Value

list of ggplot objects, each object is the plot for the corresponding
subpopulation
