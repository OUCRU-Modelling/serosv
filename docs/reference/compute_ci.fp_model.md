# Compute confidence interval for fractional polynomial model

Compute confidence interval for fractional polynomial model

## Usage

``` r
compute_ci.fp_model(x, ci = 0.95, le = 100, ...)
```

## Arguments

- x:

  \- serosv models

- ci:

  \- confidence interval

- le:

  \- number of data for computing confidence interval

- ...:

  \- arbitrary argument

## Value

confidence interval dataframe with 4 variables, x and y for the fitted
values and ymin and ymax for the confidence interval
