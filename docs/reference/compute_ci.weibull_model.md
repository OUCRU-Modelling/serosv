# Compute confidence interval for Weibull model

Compute confidence interval for Weibull model

## Usage

``` r
compute_ci.weibull_model(x, ci = 0.95, ...)
```

## Arguments

- x:

  \- serosv models

- ci:

  \- confidence interval

- ...:

  \- arbitrary argument

## Value

confidence interval dataframe with 4 variables, x and y for the fitted
values and ymin and ymax for the confidence interval
